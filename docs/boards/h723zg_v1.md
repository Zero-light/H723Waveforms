# h723zg_v1 Board Details

## Pinout

| Function | Pin | Note |
|---|---|---|
| ADC CH0 (PA6, fixed) | PA6 | ADC1_IN3, always enabled |
| ADC CH1 (PA7, optional) | PA7 | ADC1_IN7, user selectable |
| DAC OUT1 | PA4 | DAC1 channel 1 |
| SPI2 SCK | PB13 | AF5 |
| SPI2 MISO | PB14 | AF5 |
| SPI2 MOSI | PB15 | AF5 |
| SPI2 CS | PB12 | software controlled |
| Wave CH0 | PA0 | XYNC |
| Wave CH1 | PA1 | SCLK |
| Wave CH2 | PA2 | SH_R |
| Wave CH3 | PA3 | SH_S |
| Wave CH4 | PA5 | RST |
| ADC CH2 | PC1 | ADC1_INP11 |
| XYNC Snapshot | PC4 | ADC1_IN4, via dupont |
| CLK Snapshot | PB0 | ADC1_IN9, via dupont |
| LED | PG7 | active high |

## Clock Tree

- HSI 64 MHz → PLL1 → 192 MHz SYSCLK
- APB1 / APB2: 96 MHz
- ADC clock: 48 MHz (HCLK/4 via PLL2R/CLKP)

## Important notes

- PA6 must use `ADC_CHANNEL_3`, not `ADC_CHANNEL_6`.
- AF register for PA6/PA7 is cleared in `bsp_init.c` to prevent TIM/SPI leakage.
- Sampling time is set to the maximum `ADC_SAMPLETIME_810CYCLES_5`.
